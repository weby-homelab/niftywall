# 🛡️ Security Policy: NiftyWall

*Ensuring the transparency and security of your network management.*

At **Weby Homelab**, we take the security of our firewall management tools seriously. If you believe you have found a security vulnerability in **NiftyWall**, please report it to us as described below.

## Supported Versions

We currently provide security updates for the following versions of NiftyWall:

| Version | Supported | Status |
| :---: | :---: | :--- |
| **3.0.x** | ✅ | **Current (Hardened)** |
| 2.0.x | ❌ | End of Life (Legacy) |

## Reporting a Vulnerability

Please report security flaws to **[contact@srvrs.top](mailto:contact@srvrs.top)**. Do not use public GitHub issues.
